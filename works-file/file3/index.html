夢のような沖縄体験を

View

極上の沖縄体験と、忘れられない思い出を。

旅の思い出を、更なるものへ。 私たちは来沖されるお客様に向け、記憶に残るような極上のサービスと、心安らぐ室内空間を提供しております。ここでしか見ることのできない景色をさらに引き立て、お客様の沖縄への旅を忘れられない思い出へと昇華させます。

Guest Room

沖縄だけの絶景のオーシャンビュー

部屋の大きな窓から鮮やかな青い太平洋を一望することができ、沖縄特有のロケーションを思い存分味わうことができるお部屋。

more

亜熱帯の緑を全身で感じるマウンテンビュー

沖縄の亜熱帯気候ならではの自然豊かな森林の景色を一望することのできる客室。沖縄固有の植物や動物の姿を眺めることができます。運が良いと沖縄の固有種であるノグチゲラなどの姿を見れることもあります。

more

ほかの客室と一線を画す上品なスイートルーム

ホテルの最上階２室にしかない特別なお部屋。プールやサウナも完備しており、沖縄の豊かな自然を十分に満喫することのできるお部屋。お客様の望む様々な選択肢の贅沢を取り揃えています。

more

ご家族連れの方へ向けたファミリールーム

広々としたお部屋にベッドを多数取り揃えており、最大８人までご宿泊が可能です。小さなお子様を連れた方のための設備も整っております。
